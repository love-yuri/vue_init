# vue 初始化模板

## 快速开始

### 安装依赖

```sh
npm install
```

### 运行
> 如果需要实时执行taildwindcss 请执行 npx tailwindcss -i ./src/assets/main.css -o ./tailwindcss.css --watch

```sh
npm run dev
```
